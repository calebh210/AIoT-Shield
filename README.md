## Installation Steps
Requirements:
- Python3
- pip3

Installation:
```
  git clone https://github.com/calebh210/AIoT-Shield.git
  cd AIoT-Shield
  python3 -m venv .venv
  source .venv/bin/activate
  pip3 -r requirements.txt
```

Alternatively, use the installation script

```
./install.sh
```

## Usage

Launch the program
```
./main.py
```
